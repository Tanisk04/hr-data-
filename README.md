# 📊 HR Data Analysis with Python

## 📌 Overview
This project analyzes HR employee data using Python to uncover valuable insights into employee demographics, attrition, job satisfaction, and workforce trends. The analysis helps organizations make informed HR decisions through data visualization and statistical analysis.

---

## 🎯 Goal
- Analyze employee data efficiently.
- Identify factors affecting employee attrition.
- Discover workforce trends and patterns.
- Support data-driven HR decision-making.
- Present insights through clear visualizations.

---

## ✨ Features
- ✅ Data Cleaning & Preprocessing
- ✅ Exploratory Data Analysis (EDA)
- ✅ Employee Attrition Analysis
- ✅ Department-wise Analysis
- ✅ Gender-wise Analysis
- ✅ Salary & Experience Analysis
- ✅ Data Visualization using Charts
- ✅ Business Insights & Recommendations

---

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset
The dataset contains HR employee information, including:
- Employee ID
- Age
- Gender
- Department
- Job Role
- Education
- Monthly Income
- Years at Company
- Job Satisfaction
- Work-Life Balance
- Overtime
- Performance Rating
- Attrition

- Employee Attrition Analysis

- Data Visualization
